# `station_raw/private/`

Reserved for raw source files from any non-MeteoSwiss provider (SLF/IMIS,
ARPA Lombardia, ARPA Piemonte, ski-resort feeds, etc.) that turns out to
have a usable, licensed, machine-readable historical archive. As of this
session no such provider has a working adapter in this codebase - see
`docs/STATION_RESEARCH.md` for what was investigated and why nothing here
has been populated yet.
