      Known limitations

      • Historical retraining still labels against MeteoSwiss Samedan (kitesailing.ch has no historical archive), while the live loop labels against the real Silvaplana lake reading - a real, currently-open labeling-criterion mismatch.
      • Backtest features come from Open-Meteo's 0-hour archive, not a genuine 1-3 day-ahead forecast - expect live accuracy below the frozen holdout's.
      • The 2026 holdout is a partial season; trust the live accuracy number as it accumulates.
