Feature ablation

Diagnostic only — not model selection

Each row is its own model, trained fresh on 2024+2025 with only the listed features, scored once against the same 2026 holdout above. Picking the best-looking row here and reporting its holdout number as "the" model's accuracy would itself be a form of holdout leakage — this table exists to explain roughly how much each feature group contributes, not to select a winner. The deployed model always uses the full feature set.
